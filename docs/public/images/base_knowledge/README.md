# 公共基础知识课程图片

本目录存放 F1、F2 文档的本地图片，避免依赖内网链接无法访问。

## 下载方式

**在内网环境**运行以下命令即可自动下载所有图片：

```bash
node scripts/download-base-knowledge-images.mjs
```

若无法运行脚本，可从飞书文档手动保存图片并放入本目录，按以下文件名命名。
